# WeatherApp – Frontend M2

Aplicación de clima desarrollada como proyecto del Módulo 2. Muestra el clima actual de 12 ciudades chilenas con navegación a vista de detalle y pronóstico semanal.

## Tecnologías usadas

- HTML5 semántico
- Bootstrap 5 (CDN)
- JavaScript vanilla

## Cómo ejecutar

1. Clona el repositorio
2. Abre `index.html` directamente en el navegador
